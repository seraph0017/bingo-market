# 钱包模块 - 高保真Figma原型

## 设计概述
基于低保真线框图，创建符合WCAG 2.1 AA标准的高保真视觉设计，支持越南语/英语双语，移动端优先。

## 色彩系统
- **主色**: #2563EB (蓝色) - 代表信任和专业
- **辅助色**: #10B981 (绿色) - 代表成功和增长  
- **警告色**: #F59E0B (橙色) - 代表注意和提醒
- **错误色**: #EF4444 (红色) - 代表错误和危险
- **背景色**: #FFFFFF (白色) / #F9FAFB (浅灰)
- **文字色**: #1F2937 (深灰) / #6B7280 (中灰) / #9CA3AF (浅灰)

## 字体系统
- **标题**: Inter Bold, 24px, 行高 32px
- **副标题**: Inter SemiBold, 18px, 行高 24px  
- **正文**: Inter Regular, 16px, 行高 24px
- **辅助文字**: Inter Regular, 14px, 行高 20px
- **标签**: Inter Medium, 12px, 行高 16px

## 页面设计

### 1. 钱包首页
**布局结构**:
- 顶部状态栏: 时间、信号、电池
- 导航栏: 钱包图标 + "我的钱包" 标题 + 设置图标
- 余额卡片: 
  - 背景: 渐变蓝 (#2563EB → #3B82F6)
  - 圆角: 16px
  - 内边距: 24px
  - 余额显示: Inter Bold, 32px, 白色
  - 币种标识: "知识币" + VND图标
- 快捷操作区:
  - 充值按钮: 主色填充, 白色文字, 圆角8px
  - 兑换商城: 辅助色边框, 辅助色文字, 圆角8px
- 交易记录列表:
  - 列表项高度: 64px
  - 头像: 32x32px, 圆形
  - 标题: Inter SemiBold, 16px
  - 描述: Inter Regular, 14px, 浅灰色
  - 金额: Inter SemiBold, 16px, 根据类型显示绿色(收入)/红色(支出)
  - 时间: Inter Regular, 12px, 浅灰色

**交互状态**:
- 按钮悬停: 亮度增加10%
- 按钮点击: 亮度降低10%, 添加涟漪效果
- 列表项点击: 背景色变为#F3F4F6, 200ms过渡

**可访问性**:
- 色彩对比度: ≥ 4.5:1
- 触摸目标: ≥ 44x44px
- 屏幕阅读器: 所有元素都有aria-label

### 2. 充值页面
**布局结构**:
- 导航栏: 返回箭头 + "充值" 标题
- 充值金额输入:
  - 输入框: 48px高度, 圆角12px, 边框#D1D5DB
  - 占位符: "请输入充值金额"
  - 键盘类型: 数字键盘
- 快捷金额按钮:
  - 按钮尺寸: 64x40px
  - 圆角: 20px
  - 边框: 1px solid #D1D5DB
  - 文字: Inter Medium, 14px
  - 选中状态: 主色边框 + 主色文字
- 支付方式选择:
  - 卡片式布局, 8px圆角
  - 左侧图标: 24x24px
  - 右侧信息: Inter Regular, 16px
  - 选中状态: 主色左侧边框 + 背景色#F3F4F6
- 确认按钮:
  - 全宽按钮, 48px高度
  - 主色背景, 白色文字
  - 禁用状态: #9CA3AF背景, 白色文字, 50%透明度

**验证规则**:
- 最小金额: 10,000 VND
- 最大金额: 500,000 VND (日限额)
- 实时验证: 输入时即时反馈

**错误状态**:
- 错误边框: #EF4444
- 错误提示: Inter Regular, 14px, #EF4444, 显示在输入框下方

### 3. 交易记录详情
**布局结构**:
- 导航栏: 返回箭头 + "交易详情" 标题
- 交易状态卡片:
  - 成功: 绿色背景 + 对勾图标
  - 处理中: 蓝色背景 + 加载动画
  - 失败: 红色背景 + 叉号图标
- 交易信息区块:
  - 交易ID: Inter Regular, 14px, #6B7280
  - 交易时间: Inter Regular, 14px, #6B7280  
  - 交易金额: Inter Bold, 24px, 主色
  - 交易类型: Inter SemiBold, 16px
- 操作按钮:
  - 联系客服: 边框按钮, 主色边框 + 主色文字
  - 分享凭证: 图标按钮 + "分享"文字

### 4. 充值成功页面
**布局结构**:
- 对勾动画: Lottie动画, 120x120px
- 标题: "充值成功!", Inter Bold, 24px
- 描述: "您的账户已成功充值", Inter Regular, 16px, #6B7280
- 金额显示: Inter Bold, 32px, 主色
- 继续按钮: "返回钱包", 主色背景, 白色文字
- 自动跳转: 3秒后自动返回钱包首页

## 国际化适配
- **越南语文本扩展**: 所有容器预留30%额外宽度
- **日期格式**: DD/MM/YYYY
- **数字格式**: 使用本地化分隔符
- **RTL支持**: 未来扩展考虑

## 响应式断点
- **Mobile**: 320px - 414px (主要目标)
- **Tablet**: 768px - 1024px (次要支持)  
- **Desktop**: 1024px+ (基础支持)
