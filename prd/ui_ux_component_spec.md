# UI/UX 组件规范

## 设计原则
- **层级清晰**：重要信息突出显示，视觉权重引导用户注意力
- **一致性**：跨模块UI/UX保持一致，减少用户学习成本  
- **即时反馈**：用户操作后立即提供反馈，避免不确定感
- **容错设计**：防止用户犯错，提供撤销和恢复选项
- **无障碍优先**：WCAG 2.1 AA 合规，确保所有用户都能使用

## 核心组件规范

### 按钮系统
| 变体 | 用途 | 颜色 | 状态 |
|------|------|------|------|
| Primary | 主要操作 | --primary-500 | hover: --primary-600, active: --primary-700 |
| Secondary | 次要操作 | --gray-200 | hover: --gray-300, active: --gray-400 |
| Tertiary | 低强调操作 | transparent | hover: --gray-100, active: --gray-200 |
| Destructive | 危险操作 | --error-500 | hover: --error-600, active: --error-700 |

**尺寸**：
- Small: height 32px, padding 0 --space-3
- Medium: height 40px, padding 0 --space-4  
- Large: height 48px, padding 0 --space-5

**无障碍要求**：
- 最小触摸目标：44x44px
- 键盘可聚焦
- 屏幕阅读器支持（aria-label）

### 表单元素
**输入框**：
- 高度：40px (Medium)
- 边框：1px solid --gray-300
- 圆角：6px
- 焦点状态：2px solid --primary-500, outline: none
- 错误状态：border-color: --error-500

**选择器**：
- 下拉箭头：右侧12px位置
- 选项高度：36px
- 多选支持

**标签**：
- 字重：--font-medium
- 颜色：--gray-700
- 必填标识：红色星号 *

### 状态指示器
**加载状态**：
- Spinner动画
- 文字提示："处理中..."

**成功状态**：
- 图标：✓ + 绿色文字
- 自动消失：3秒

**错误状态**：
- 图标：✗ + 红色文字  
- 持续显示直到用户操作

**空状态**：
- 插图 + 描述文字 + CTA按钮
- 示例："暂无交易记录\n开始你的第一次预测吧！"

### 导航组件
**标签页**：
- 激活状态：底部2px --primary-500边框
- 悬停效果：轻微背景色变化

**面包屑**：
- 分隔符：/ 或 >
- 当前页：--gray-500颜色

**分页器**：
- 当前页：--primary-500背景
- 禁用状态：--gray-200背景

## 国际化考虑
- **文本扩展**：越南语文本比英语长约20-30%，预留足够空间
- **数字格式**：越南使用 . 作为千分位分隔符，, 作为小数点
- **日期格式**：DD/MM/YYYY
- **货币格式**：VND（越南盾），符号在数字后
- **RTL支持**：虽然越南不需要，但代码结构支持未来扩展

## 无障碍要求 (WCAG 2.1 AA)
- **色彩对比度**：文本与背景 ≥ 4.5:1，大文本 ≥ 3:1
- **键盘导航**：所有交互元素可通过Tab键访问
- **焦点指示器**：清晰可见的焦点状态
- **屏幕阅读器**：
  - 语义化HTML结构
  - ARIA标签和角色
  - 动态内容更新通知
- **字体缩放**：支持系统字体缩放，布局不破坏

## 交互模式
### 移动端优化
- **拇指友好区域**：主要操作按钮位于屏幕下半部分
- **防误触**：相邻按钮间距 ≥ 8px
- **手势支持**：左滑删除、下拉刷新等

### 桌面端优化  
- **悬停效果**：提供额外信息和操作选项
- **快捷键**：常用操作支持键盘快捷键
- **多窗口**：支持多标签页同时操作

## 动画和过渡
- **微交互**：按钮点击、状态切换有轻微动画
- **页面过渡**：平滑的页面切换效果
- **加载动画**：骨架屏或进度条
- **持续时间**：200-300ms，不过度干扰用户

## 错误处理
### 表单验证
- **实时验证**：输入时即时反馈
- **提交验证**：提交时全面检查
- **错误消息**：具体、可操作的指导

### 系统错误
- **网络错误**：重试按钮 + 离线缓存
- **服务错误**：友好错误页面 + 联系支持
- **数据错误**：自动修复或明确指引

## 性能优化
- **懒加载**：图片和非关键内容延迟加载
- **代码分割**：按路由分割代码包
- **缓存策略**：静态资源长期缓存
- **首屏优化**：关键CSS内联，非关键JS异步

## 测试要求
- **跨浏览器**：Chrome、Safari、Firefox最新2版本
- **跨设备**：iOS 12+、Android 8+
- **无障碍测试**：屏幕阅读器、键盘导航
- **性能测试**：Lighthouse评分 ≥ 90
