## 🚀 快速启动

### 后端启动
```bash
cd code/backend
/Users/xunan/miniconda3/envs/bingo/bin/python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

访问 API 文档：http://localhost:8000/docs

### 前端启动
```bash
cd code/frontend
npm run dev
```

访问前端：http://localhost:5173

## 📁 项目结构

```
bingo-market/
├── code/
│   ├── backend/              # Python FastAPI 后端
│   │   ├── app/
│   │   │   ├── main.py       # 应用入口
│   │   │   ├── api/v1/       # API 路由 (18 个模块)
│   │   │   ├── models/       # SQLAlchemy 模型 (26 张表)
│   │   │   ├── schemas/      # Pydantic 模式
│   │   │   ├── services/     # 业务逻辑 (18 个服务)
│   │   │   └── core/         # 配置/安全/数据库
│   │   └── requirements.txt
│   └── frontend/             # Vue 3 前端
│       ├── src/
│       │   ├── views/        # 页面组件 (10 个)
│       │   ├── components/   # 可复用组件
│       │   ├── stores/       # Pinia 状态管理
│       │   ├── utils/        # 工具函数
│       │   └── i18n.ts       # 多语言配置
│       └── package.json
├── prd/                      # 产品需求文档 (8 个模块)
├── design/                   # UI/UX 设计规范
└── CLAUDE.md                 # 项目开发指南
```

## 🔧 技术亮点

### 架构设计
- **CQRS**: 读写分离设计
- **Saga**: 跨模块事务处理
- **断路器**: 服务降级保护

### 安全合规
- JWT 双令牌机制 (Access + Refresh)
- 18+ 实名认证
- 设备指纹追踪
- 审计日志全覆盖
- 越南合规限额

### 用户体验
- 三语言支持 (越南语/英语/中文)
- 响应式设计 (Mobile-first)
- WCAG 2.1 AA 无障碍标准
- 实时市场概率更新
