# 产品设计原则

## 核心设计原则

### 1. 层级原则 (Hierarchy)
- **视觉权重引导注意力**：重要信息使用更大的字体、更醒目的颜色
- **尺寸、颜色、对比度指示重要性**：关键操作按钮使用主色，次要操作使用中性色
- **相关元素分组**：使用间距和容器将相关功能组合在一起

### 2. 一致性原则 (Consistency)
- **重用模式和组件**：相同功能使用相同的UI模式
- **保持视觉节奏**：统一的间距系统（4px基准单位）
- **可预测的交互**：用户能预知操作结果

### 3. 反馈原则 (Feedback)
- **确认用户操作**：按钮点击后立即显示加载状态
- **显示系统状态**：交易处理中、审核中等状态清晰标识
- **指示加载状态**：长时间操作显示进度条或加载动画

### 4. 容错原则 (Error Prevention & Recovery)
- **防止用户犯错**：敏感操作前二次确认
- **提供撤销选项**：支持取消未完成的操作
- **清晰的错误提示**：错误信息具体且提供解决方案

### 5. 可访问性原则 (Accessibility)
- **色彩对比度**：文本与背景对比度至少4.5:1
- **焦点指示器**：键盘导航时清晰显示焦点状态
- **屏幕阅读器支持**：所有交互元素有适当的ARIA标签
- **触摸目标大小**：最小触摸目标44x44px

--color-gray-500: #6b7280;

### 字体系统
```css
/* 字体大小 */
--text-xs: 0.75rem;    /* 12px */

--text-base: 1rem;     /* 16px */
--text-lg: 1.125rem;   /* 18px */
--text-xl: 1.25rem;    /* 20px */
--text-2xl: 1.5rem;    /* 24px */

/* 行高 */
--leading-tight: 1.25;
--leading-normal: 1.5;
--leading-relaxed: 1.75;
```

### 间距系统 (4px基准)
```css
--space-1: 0.25rem;   /* 4px */
--space-2: 0.5rem;    /* 8px */
--space-3: 0.75rem;   /* 12px */
--space-4: 1rem;      /* 16px */
--space-5: 1.25rem;   /* 20px */
--space-6: 1.5rem;    /* 24px */
--space-8: 2rem;      /* 32px */
```

### 组件规范

#### 按钮系统
- **Primary**: 主要操作（充值、购买、提交）
- **Secondary**: 次要操作（取消、返回）
- **Tertiary**: 低强调操作（查看更多、编辑）
- **Destructive**: 破坏性操作（删除、冻结）

#### 表单元素
- **输入框**: 圆角4px，内边距8px，聚焦时蓝色边框
- **选择器**: 下拉箭头，选项悬停高亮
- **复选框/单选**: 20x20px，选中状态主色填充

#### 状态指示器
- **Loading**: 旋转动画，浅灰色背景
- **Success**: 绿色对勾图标，成功消息
- **Error**: 红色叉号图标，具体错误描述
- **Warning**: 黄色感叹号，警告说明

## 国际化设计规范

### 文本扩展率
- **越南语**: 预留30%额外空间（比英语长20-30%）
- **英语**: 基准长度
- **按钮文本**: 最大字符数限制（越南语20字符，英语15字符）

### 本地化格式
- **日期格式**: DD/MM/YYYY HH:mm
- **数字格式**: 千分位分隔符（越南：1.000.000）
- **货币格式**: VND（越南盾），无小数位
- **方向**: LTR（从左到右）

## 可访问性检查清单

### WCAG 2.1 AA 合规
- [ ] 所有文本对比度 ≥ 4.5:1
- [ ] 所有交互元素可通过键盘访问
- [ ] 所有图片有alt文本
- [ ] 表单有明确的标签
- [ ] 错误信息可被屏幕阅读器读取
- [ ] 动画可被用户禁用
- [ ] 触摸目标 ≥ 44x44px

### 屏幕阅读器支持
- [ ] 按钮有aria-label（当只有图标时）
- [ ] 表单字段有aria-describedby（错误信息）
- [ ] 状态变化有aria-live区域
- [ ] 导航有landmark区域（header, main, footer）

### 关键页面模板

#### 表单页面
- 顶部：页面标题和说明
- 中部：表单字段（垂直排列）
- 底部：主要操作按钮（固定位置，便于拇指操作）

#### 列表页面  
- 顶部：筛选器和搜索
- 中部：列表项（卡片式布局）
- 底部：分页或"加载更多"

#### 详情页面
- 顶部：返回按钮和标题
- 中部：详细信息展示
- 底部：相关操作按钮

## 交互模式

### 加载状态
- **即时反馈**: 按钮点击后立即显示加载状态
- **骨架屏**: 数据加载时显示占位符
- **进度指示**: 长时间操作显示进度条

### 错误处理
- **内联验证**: 表单字段实时验证
- **错误汇总**: 提交失败时显示所有错误
- **恢复建议**: 错误信息包含解决方案

### 成功反馈
- **Toast通知**: 操作成功后短暂显示
- **页面跳转**: 重要操作后跳转到结果页面
- **状态更新**: 相关数据立即刷新
