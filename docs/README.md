## 欢迎

欢迎来到 **Bingo Market** 技术文档中心。本文档中心提供项目完整的技术文档，包括产品需求、技术规格、API 文档、测试用例和运维指南。

**Bingo Market** 是一个面向东南亚市场的合规预测平台，首发越南市场，聚焦科技、商业、文化和学术知识领域。

### 合规声明

本平台严格遵守越南市场法规：
- 18+ 实名认证
- 日限额 500K VND，月限额 5M VND
- 单向充值（不支持提现/转账）
- 禁止体育比分、政治选举、宗教话题

## 快速开始

### 开发环境搭建

#### 后端启动

```bash
cd code/backend

# 安装依赖
/Users/xunan/miniconda3/envs/bingo/bin/python -m pip install -r requirements.txt

# 配置环境
cp .env.example .env
# 编辑 .env 配置数据库和密钥

# 启动服务
/Users/xunan/miniconda3/envs/bingo/bin/python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

访问 API 文档：http://localhost:8000/docs

#### 前端启动

```bash
cd code/frontend

# 安装依赖
npm install

# 配置环境
cp .env.example .env

# 启动开发服务器
npm run dev
```

访问前端应用：http://localhost:5173

## 文档规范

### 文档编写

所有技术文档应遵循 [文档编写规范](./DOCUMENTATION_GUIDE.md)，包括：

- **标准结构**：元数据、修订历史、目录、正文
- **格式规范**：标题层级、代码块、表格、列表
- **术语规范**：使用合规术语，避免敏感词汇
- **版本控制**：语义化版本号，完整修订历史

### 合规术语

| 使用 | 不使用 | 说明 |
|------|--------|------|
| 预测 | 赌博/下注 | 平台性质定义 |
| 知识币 | 筹码/代币 | 平台虚拟货币 |
| 份额 | 赌注 | 用户持仓单位 |
| 市场共识概率 | 赔率/胜率 | 价格显示方式 |
